# Blog em NodeJS

## Sobre
Blog interativo com conteúdos de direito. Permite a autenticação de usuários, visualização e salvamento de publicações.

## Tecnologias utilizadas
* Express v4.18.1
* Express-Handlebars v6.0.6
* Bootstrap v5.2.2
